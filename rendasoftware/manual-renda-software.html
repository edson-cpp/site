6. Abra sua Conta no PagSeguro

O PagSeguro é o sistema de Pagamentos e Recebimentos On-Line da UOL 100% seguro como o nome diz. Por meio dele você pode receber pagamentos via cartão de crédito, transferência on-line e boleto bancário. O Renda Software usa o PagSeguro como meio oficial para você receber seus pagamentos pela senha de registro do programa. Portanto, antes de personalizar o seu Renda Software, você precisa ter uma conta do tipo Vendedor aberta no PagSeguro.

Acesse o site do PagSeguro pelo endereço abaixo:

www.pagseguro.uol.com.br

 

Clique no botão "CADASTRE-SE JÁ" e faça o cadastro gratuitamente preenchendo seus dados

 

7. Modificando sua Conta PagSeguro para Vendedor

Agora que você já tem uma conta aberta no PagSeguro, deve mudá-la para o tipo Vendedor. Esse procedimento é simples e não tem nenhum custo extra.

 

O PagSeguro carregará o formulário de alteração de conta no final da página, conforme a figura seguinte. Preencha  os dados solicitados.

  • No campo "Tipo de comércio", escolha "Indústria, Comércio e Agronegócio", ou "Informática". 

  • No campo "URL do seu site", informe o endereço do seu site. Se você ainda não tem um site, deixe este campo em branco.

  • Os campos "URL de Leilão 1", "URL de Leilão 2" e "Apelido de Leilão" podem ficar em branco. Se você costuma vender produtos em sites de Leilão e pretende aceitar pagamentos via PagSeguro nessas vendas, pode preencher esses campos.

  • No item "Local de Vendas", selecione "Casa / Escritório". Se você tem um site de vendas, marque também a opção "Site próprio".

    ATENÇÃO: Note que, se você marcar "Leilões OnLine" no item "Local de Vendas", terá de preencher obrigatoriamente pelo menos a "URL de Leilão 1" e o primeiro "Apelido de Leilão". E se você marcar a opção "Site próprio", terá de preencher obrigatoriamente o campo 'URL do seu site".

 

8. Verificando sua Conta PagSeguro

Modifique sua conta Pagseguro para receber pagamentos via cartões de crédito também

Para aceitar todos os tipos de pagamento, a sua conta PagSeguro precisa ser verificada . Isso significa que alguns de seus dados devem ser validados para evitar fraudes. 

 

 

Você deverá enviar copia de comprovante de residência, cpf e identidade para o pagseguro.

 

Verificação de Endereço: Clique no botão "VERIFICAR" deste item. O sistema do PagSeguro fará a validação do endereço que você forneceu por ocasião do sesu cadastro e, se tudo estiver certo, você receberá a mensagem:

Seus dados foram verificados com sucesso e o status de sua conta foi atualizado para 'Verificado'.

Agora sua conta PagSeguro já poderá ser configurada para aceitar pagamentos por todos os meios disponívels!

9. Configurando Sua Conta Para Aceitar Cartões de Crédito

15. Criando o seu Mini-Site para Distribuir

Você precisará de uma página de vendas para este site, exatamente igual a minha cybersis.com.br/rendasoftware

Criar um site, mesmo que seja pequeno e simples, normalmente exige algum conhecimento técnico. Mas no caso do Renda Software você não precisa ter esse conhecimento. Isso porque o Renda Software lhe oferece como bônus um Gerador de Mini-Site. Siga os passos seguintes para gerar de modo muito fácil o seu próprio mini-site de distribuição do Renda Software.

Download do Gerador de Mini-Sites

Se o seu Windows Explorer não estiver aberto, abra-o e localize a pasta onde se encontra o seu Renda Software, provavelmente em "C:\rendasoftware". Abra novamente o arquivo "rendasoftware.exe" com um duplo clique do mouse sobre ele, conforme mostra a figura abaixo.

Clique no link "Bônus"

Na tela de Bônus, clique no link "Gerador de Mini-Sites".

ATENÇÃO: Você precisa estar conectado na Internet para baixar o Gerador de Mini-Sites.

Salve o Gerador de Mini-Sites  na mesma pasta onde salvou o seu Renda Software.

 

Descompacte o Gerador de Mini-Sites

 

Gerando o seu Mini-Site Renda Software com o Gerador

 

Preencha o formulário abaixo com seus dados

Preencha o formulário com todo o cuidado. Os campos solicitados são os seguintes:

  1. URL (link) que os seus visitantes usarão para fazer o download do seu Renda Software: Neste campo você deve colocar o endereço do site que você criou no Passo 7, mais o nome do arquivo "rendasoftware.zip". Este será o endereço completo de download do seu Renda Software. Por exemplo, no meu caso eu criei o site http://cybersis.com.br/rendasoftware, portanto o endereço de download do meu Renda Software neste campo vai ficar assim:

          http://cybersis.com.br/rendasoftware/rendasoftware.zip 

    OBS: Você deve colocar uma barra "/" separando o endereço principal do seu site e o nome do arquivo "rendasoftware.zip".

     
  2. Digite seu nome completo.

  3. Digite seu endereço residencial: Neste campo eu não recomendo colocar o seu endereço completo, pois é uma exposição desnecessária. Sugiro que coloque apenas sua cidade e estado. Exemplo: "São Paulo - SP".

  4. Digite seu endereço de e- mail.

  5. Digite a URL do seu website: Neste campo você deve colocar o endereço principal do site que você criou no Passo 7 (com http:// antes). No meu caso, o endereço é http://cybersis.com.br/rendasoftware

     

 

Na hora de salvar o seu mini-site, localize a mesma pasta onde estão os demais arquivos do seu Renda Software (C:\rendasoftware, se você fez como eu sugeri). No campo "Nome do arquivo", escreva "index" (sem aspas e sem espaços) e clique no botão "Salvar".

O Gerador de Mini-Sites vai gerar o seu mini-site Renda Software e vai gravar os seguintes 3 arquivos na sua pasta "rendasoftware":

  • index.html: É a página principal do seu mini-site Renda Software.

  • rendasoftheader.jpg: É a figura do cabeçalho do seu mini-site.

  • rendasoftbox.jpg: É a figura da "caixa" do Renda Software.

Se você fizer uma boa divulgação do seu mini-site Renda Software na Internet, certamente começará em breve a receber pagamentos pela senha em sua conta PagSeguro.

Cada vez que você receber um pagamento referente a senha do Renda Software e esse pagamento for aprovado, você deverá enviar a senha do Renda Software para o e-mail do comprador. Vamos entender melhor como isso funciona.

Quando alguém iniciar um pagamento a você pelo PagSeguro, você receberá um e-mail do PagSeguro lhe notificando desse pagamento. Porém, isso não significa que você já pode mandar a senha para a pessoa! Somente quando o pagamento aparecer o status Aprovado você poderá enviar a senha.
